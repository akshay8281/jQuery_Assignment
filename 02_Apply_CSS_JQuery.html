2. How to Apply CSS Using JQuery.

Ans -

In this Example

  • Include the jQuery library using a script tag.
  • Create a button and a paragraph in the HTML body.
  • Use a script block to write jQuery code. Inside the
    $(document).ready() function, handle a button click event using $("#applyStylesBtn").click().
  • Use $("p").css() to apply CSS styles to the paragraph element when the button is clicked. You can specify the styles as key-value pairs within the curly braces.

In this case, when you click the "Apply Styles" button, the paragraph will have a light blue background, dark blue text color, and a font size of 18 pixels. Adjust the styles according to your needs.
