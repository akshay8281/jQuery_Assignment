03. How to Add Class and Remove Class in Jquery

Ans -

Certainly! Here's a simple HTML program that demonstrates how to use jQuery to add and remove classes from an element:

  • Two buttons (Add Class and Remove Class) and a paragraph (

    ) element are created in the HTML body.

  • CSS class .highlighted is defined with some styling properties.
  • The jQuery library is included using a script tag.
  • jQuery code inside $(document).ready() function adds and removes the class when buttons are clicked.
    • $("#addClassBtn").click() adds the class "highlighted" to the paragraph when the "Add Class" button is clicked.
    • $("#removeClassBtn").click() removes the class "highlighted" from the paragraph when the "Remove Class" button is clicked.

This is a simple demonstration, and you can modify it to suit your specific requirements. When you click the buttons, you'll see the paragraph's background color and font weight change due to the addition and removal of the "highlighted" class.
